# Snake Water Gun

A simple command-line game built with Python based on the classic **Snake, Water, Gun** game. The player competes against the computer, and the winner is determined according to the game's rules.

## Features

- Random computer moves
- Interactive gameplay
- Score tracking (if implemented)
- Beginner-friendly implementation
- Clean and readable code


## How to Run

```bash
python main.py
```

## Project Structure

```
SnakeWaterGun/
├── main.py
└── README.md
```

## Purpose

This project was created to practice Python fundamentals such as loops, conditional statements, functions, user input, and the `random` module while building an interactive command-line game.
