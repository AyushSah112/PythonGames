# Cow Bull Game

A simple command-line game built with Python where the player tries to guess a secret number. After each guess, the game provides feedback using **Cows** and **Bulls** to help the player identify the correct number.

## Features

- Randomly generated secret number
- Cow and Bull hint system
- User input validation
- Beginner-friendly implementation
- Clean and readable code


## How to Run

```bash
python main.py
```

## Project Structure

```
CowBullGame/
├── main.py
└── README.md
```

## Purpose

This project was created to practice Python fundamentals such as loops, conditional statements, string manipulation, user input, and problem-solving by building an interactive guessing game.
